# Docs Guide

이 디렉토리는 루트 디렉토리를 단순하게 유지하기 위해 보조 문서를 모아 둔 곳입니다.

## 폴더 설명

| 폴더 | 내용 |
|---|---|
| `architecture/` | 현재 코드 계층과 데이터 파이프라인 다이어그램 |
| `project/` | 프로젝트 규칙, 로드맵, 기획 문서, 코드 리뷰 규칙 |
| `logs/` | 작업별 구현 로그와 실행 결과 |
| `study/` | 혼자 공부할 개념과 기술 목록 |
| `prompts/` | 에이전트 작업용 프롬프트 |
| `references/` | API 활용가이드 등 외부 참고 자료 |
