# Final Report

## 제목

공공 금융 데이터로 주식 방향성 예측 ML 파이프라인 만들기

## 요약

본 프로젝트는 공공데이터포털과 KRX 등 공개 금융 데이터를 활용해 주식 일별 시세와 시장지수를 수집하고, 향후 5영업일 수익률이 양수인지 예측하는 이진 분류 모델을 구현한다. 핵심은 고성능 모델보다 데이터 누수 방지, 시간 순서 기반 검증, 단순 백테스트를 통한 모델 한계 분석이다.

## 작성 예정 항목

- 데이터 출처
- feature 목록
- label 정의
- 검증 방식
- 모델 비교
- 백테스트 결과
- 한계점
- 다음 개선 방향
